Calculateur d'Intérêt Simple
Description
Ce projet est un calculateur d'intérêt simple développé avec HTML, CSS et JavaScript. Il permet aux utilisateurs de calculer facilement les intérêts simples en fonction du montant principal, du taux d'intérêt annuel et de la durée.
Fonctionnalités

Interface utilisateur intuitive et responsive
Calcul instantané des intérêts simples
Validation des entrées utilisateur
Affichage clair des résultats
Compatible avec tous les navigateurs modernes

Formule Utilisée
L'intérêt simple est calculé avec la formule suivante :
Intérêt Simple = (Principal × Taux × Temps) / 100
Où :

Principal : Le montant initial investi ou emprunté
Taux : Le taux d'intérêt annuel (en pourcentage)
Temps : La durée de l'investissement ou du prêt (en années)

Installation

Clonez le dépôt :

bashgit clone https://github.com/votre-username/simple-interest-calculator.git

Accédez au répertoire du projet :

bashcd simple-interest-calculator

Ouvrez le fichier index.html dans votre navigateur préféré.

Utilisation

Entrez le montant principal dans le champ approprié
Saisissez le taux d'intérêt annuel (en pourcentage)
Indiquez la durée en années
Cliquez sur le bouton "Calculer"
Le résultat s'affiche instantanément

Exemple
Entrées :

Principal : 10 000 €
Taux : 5%
Durée : 3 ans

Résultat :

Intérêt Simple = (10 000 × 5 × 3) / 100 = 1 500 €
Montant Total = 10 000 + 1 500 = 11 500 €
